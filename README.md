# vue-blog-admin
A beautiful back management system based on Vue

## Online preview

[线上地址](https://konglingwen94.github.io/vue-blog-admin/dist/index.html)


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
